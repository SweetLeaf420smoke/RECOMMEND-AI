# Реестр артефактов продукта (нумерованный список)

Полный набор для зрелого продукта часто **40–70** документов; стартуют обычно с **5–6**: Vision, Problem, Value proposition, PRD, Architecture, Roadmap.

Ниже — объединённый нумерованный список из переписки (можно дорабатывать по мере работы).

## Стратегия и продукт

1. One-liner (одна фраза: что это за продукт)  
2. Elevator pitch (≈30 секунд)  
3. Vision document  
4. Product principles (что всегда / никогда)  
5. Problem statement  
6. Jobs-to-be-Done (JTBD)  
7. Value proposition  
8. Positioning statement  
9. Competitive landscape (конкуренты + замены)  
10. Differentiation / Moat hypothesis  
11. Assumptions log  
12. Hypotheses backlog  
13. Risks & mitigations  
14. Success metrics / North Star metric  
15. KPI tree  
16. Measurement plan  
17. Analytics taxonomy (события/параметры)  
18. Experimentation plan (A/B рамка)  
19. Research plan
