# Рыночные сигналы и соседи по идее (выдержка из старой переписки)

**Назначение:** зафиксировать полезное из раннего диалога (питч Synq, Reddit) без превращения в источник истины. Цифры, фичи и URL **перепроверяй** перед питчем.

## Боль пользователей (Reddit, тема «найти людей со схожим вкусом в Spotify»)

- Людям не хватает способа находить **«музыкальных двойников»**, смотреть чужие плейлисты и общаться по вкусу.
- Часто звучит мысль, что **поиск и соцслой у Spotify слабые**, плейлисты сложно найти, выдача завязана на подборки платформы.
- Вывод для продукта: спрос на **внешний слой** «по вкусу» есть; конкурируешь не только с идеей, а с **привычкой** и с готовыми обходными сервисами.

## Сервисы, которые упоминались как обходной путь (проверять актуальность)

| Имя | Заметка (кратко) |
|-----|------------------|
| **stats.fm** (ранее Spotistats) | Статистика прослушиваний, сравнение, соц. функции вокруг Spotify — часто советуют в обсуждениях. |
| **playlost.fm** | Подбор похожих **пользовательских** плейлистов по пересечениям артистов/треков/жанров. |
| **Airbuds** | Социальный слой вокруг того, что слушают друзья (виджеты, реакции) — другая ось, не «глобальный матч». |
| **itsdisco** | Упоминался как проект «social for Spotify» — судить по текущему сайту/аппу отдельно. |

Это не «список конкурентов на века», а **ориентиры**, почему пользователю может быть не хватать Spotify alone.

## Простая формулировка для пользователя (из старого питча)

> Сервис объединяет интересы из разных приложений и помогает находить **людей, места и события** под твой вкус — в том числе в другой стране.

Рядом с этим держим текущую нить **TasteCompass**: сначала **культурный навигатор** (места/события), слой «людей» — осознанно позже.

## Связь с репозиторием

- **Synq** в коде — ветка **Spotify OAuth + профиль + топ-артисты + матчи** (как в раннем ТЗ).
- **TasteCompass** — отдельная траектория и демо `/taste-compass/` (опрос + каталог + правила), см. [taste-compass/README.md](./taste-compass/README.md).

Два направления могут сходиться позже (например, теги жанров из Spotify → усиление профиля вкуса), но это **не обязательный** первый шаг для навигатора.
