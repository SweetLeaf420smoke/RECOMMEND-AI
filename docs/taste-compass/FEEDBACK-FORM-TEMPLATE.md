# Шаблон сбора обратной связи (фаза A)

Для каждого предложенного места или события попроси ответ **одной строкой** или в таблице.

| № | Название места / события | Пошли бы? (да / скорее да / нет) | Почему (одно предложение) |
|---|--------------------------|-----------------------------------|---------------------------|
| 1 |                          |                                   |                           |
| 2 |                          |                                   |                           |

Дополнительно (опционально):

- «Что бы ты добавил в опрос, чтобы попало точнее?»  
- «Один пример идеального вечера в этом городе» (свободный текст)

---

Минимум для закрытия фазы A: **5 заполненных строк** по разным пунктам (не обязательно от 5 разных людей, но лучше 3–5 разных людей).
