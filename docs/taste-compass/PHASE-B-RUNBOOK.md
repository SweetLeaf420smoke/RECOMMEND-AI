# Фаза B — схема вкуса + каталог + повтор цикла

## Что уже должно быть

- Зафиксированы поля профиля: [TASTE-SCHEMA.md](./TASTE-SCHEMA.md)  
- Зафиксированы поля карточек: [CATALOG-FIELDS.md](./CATALOG-FIELDS.md)  
- Есть хотя бы черновик таблицы мест/событий (Google Sheets / Notion / JSON).

## Повтор ручного цикла для нового человека

1. Новый респондент проходит тот же [SURVEY-TEMPLATE.md](./SURVEY-TEMPLATE.md).  
2. Ты заполняешь профиль по схеме (теги не выдумывай — только из ответов).  
3. Подбираешь 5–10 пунктов **только из каталога** (если чего-то нет — добавь карточку в каталог, потом рекомендуй).  
4. Собираешь короткую обратную связь по [FEEDBACK-FORM-TEMPLATE.md](./FEEDBACK-FORM-TEMPLATE.md).

**Критерий готовности фазы B:** подборка для **нового** человека занимает **10–15 минут** по шаблону, без хаоса.

Дальше — минимальный продукт: опрос и выдача из базы в приложении (`/taste-compass`).
