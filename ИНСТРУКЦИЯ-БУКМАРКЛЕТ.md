# 🎵 Stats.fm → Spotify: Букмарклет

## Что это?
Кнопка в браузере, которая копирует все треки со страницы stats.fm в буфер обмена.

## Как установить:

### Шаг 1: Создай закладку
1. В Chrome нажми **Cmd+D** (создать закладку)
2. Назови её: `📋 Copy Stats.fm Tracks`
3. Сохрани

### Шаг 2: Замени URL закладки
1. Открой закладки: **Cmd+Shift+O**
2. Найди свою закладку
3. Нажми **Edit** (три точки справа)
4. В поле **URL** вставь этот код:

```javascript
javascript:(function(){const tracks=[];document.querySelectorAll('a[href*="/track/"]').forEach(link=>{const text=link.textContent.trim();if(text&&text.length>3&&text.length<200){tracks.push(text)}});const unique=[...new Set(tracks)];if(unique.length===0){alert('Треки не найдены! Попробуй прокрутить страницу вниз.');return}const text=unique.join('\n');navigator.clipboard.writeText(text).then(()=>{alert('✅ Скопировано '+unique.length+' треков!\n\nТеперь:\n1. Открой Spotify\n2. Создай плейлист\n3. Вставь (Cmd+V)')})})();
```

## Как использовать:

1. Открой любую страницу **stats.fm/username/tracks**
2. Прокрути вниз чтобы загрузились все треки
3. **Нажми на закладку** `📋 Copy Stats.fm Tracks`
4. Появится alert: "✅ Скопировано N треков"
5. Открой **Spotify** → создай плейлист
6. В поле поиска плейлиста нажми **Cmd+V** (вставить)
7. Готово! 🎵

---

## Альтернатива: Ручное копирование

Если букмарклет не работает:

1. Открой **DevTools** (F12)
2. Вставь в Console:

```javascript
const tracks = [];
document.querySelectorAll('a[href*="/track/"]').forEach(link => {
  tracks.push(link.textContent.trim());
});
copy([...new Set(tracks)].join('\n'));
alert('Скопировано!');
```

3. Нажми Enter
4. Треки скопированы в буфер!
